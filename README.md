# beats

Tracks made with [Lattice](https://github.com/Polyoxy/lattice) — a
theory-constrained beat engine. Each folder is a track's source:
`beat.json` regenerates it exactly, `beat.mid` opens anywhere. Audio
lives in [Releases](../../releases).
