# beats

Tracks made with [Lattice](https://github.com/Polyoxy/lattice) — a
theory-constrained beat engine. Each folder is a track's source:
`beat.json` regenerates it exactly, `beat.mid` opens anywhere. Audio
lives in [Releases](../../releases).

## seed 42 tape · 2026-07-16

**The Lion and the Fox** — the four-chord play: charm in the 6/9, teeth
in the minor five. `C#m7 → Amaj7 → F#m6add9 → G#m7`

**Play the Man** — starts on the four and circles the throne, never asks
permission to sit. `F#m7 → C#m7 → Amaj7`

**Name on the Door** — two chords. The door already says everything.
`Amaj7 → C#m7`

All three: faiyaz card, C# minor, 72 bpm, one seed.
